# Operadores:

```csharp
>  -> maior
<  -> menor
>=  -> maior ou igual
<=  -> menor ou igual
==  -> igual
!=  -> diferente

```

## Exemplos práticos

```csharp
int a = 10;
int b = 5;

Console.WriteLine(a == b);// false
Console.WriteLine(a != b);// true
Console.WriteLine(a > b);// true
Console.WriteLine(a < b);// false
Console.WriteLine(a >=10);// true
Console.WriteLine(b <=3);// false
```

## Comparando strings

```csharp
string nome = "João";

Console.WriteLine(nome =="João");// true
Console.WriteLine(nome !="Ana");// true

```
